# Floresta GUI

A cross-platform graphical interface for [Floresta](https://github.com/getfloresta/Floresta) - a lightweight Bitcoin client with Utreexo.


## Main Commands

### Development

```bash
# Install dependencies
pnpm install

# Run in development mode (hot reload)
pnpm run tauri dev

# Frontend only (without Tauri)
pnpm run dev
```

### Build

```bash
# Production build (generates native executable)
pnpm run tauri build

# Frontend build only
pnpm run build
```


### Project Setup

```bash
# Clone the repository
git clone https://github.com/lucasdbr05/floresta-gui.git
cd floresta-gui

# Install dependencies
pnpm install

# Run in development
pnpm run tauri dev
```
